## Project Description
This project demonstrates Unit Testing and Integration Testing using Python and Pytest.
A simple banking application is implemented with features such as deposit, withdraw,
interest calculation, loan eligibility check, and balance transfer.

Unit tests validate individual functions independently, while integration tests verify
that multiple functions work together correctly.

## Technologies Used
- Python 3
- Pytest
- Pytest-HTML
